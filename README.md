## Installation

1. **Clone the repository:**

   ```bash
   git clone https://github.com/vlad-burthev/JSN-TASK-server.git
   cd client
   npm install
   ```

2. **Create a .env file:**
  ```bash
  VITE_BASE_API_URL=http://localhost:8888/api/
  VITE_BASE_IMAGE_URL=http://localhost:8888/
  ```

3. **Running the Server**
   ```bash
   npm run dev
   ```
